# HL Education +

用于完全接管 ClassIsland 的每日课表选择，并按自定义上课、休息周期运行。当前版本为 `2.1.0.2`，课表、定时关机和值日生只需安装一个 `.cipx`。

## 首次使用与接管警告

- 首次启动会显示新手引导，并明确提示 HL Education + 会覆盖 ClassIsland 原本的每日课表选择。
- 用户确认前不会接管课表；选择“暂不启用”会保留 ClassIsland 当前课表状态。
- 确认后会直接打开 HL Education + 设置页，引导用户管理周次、校准今天周次并绑定班级批次时间表。
- 设置页始终显示接管警告，也可随时重新打开新手引导。
- 插件只覆盖当天课表选择，不会删除原有课表内容；暂停 HL Education + 会恢复接管前状态。

## 长周期课表

- 插件沿用原有托管课表和旧版托管时间表，从 `1.6.0.0` 升级时不会重建现有周一至周N课程或时间点。
- 原课表编辑器会替换为可横向滚动的周期编辑器，可创建周十一、周十二以及更多周次，最多 100 个上课日。
- 每个周次都可“前插”“后插”或“归档”；末尾也可直接新增。
- 插入项复制前一天的完整课表；在周一前插入时复制原周一。
- 归档不会删除课表内容。恢复时按归档记录的原位置插入，并自动重新编号为周一、周二……周N。
- 插入、归档或恢复时，插件会保持今天原本对应的课表；若今天原定休息，则保持今天在连续休息段中的序号。

## 周期校准

- 在原课表周期编辑器顶部选择“今天是周几”并确认后，后续日期会从该位置连续推算。
- 应用设置中仍保留“今天是整个周期第几天（含休息）”，用于直接校准休息日位置。
- 永久校准会清除正在执行的临时休息重复。
- 连续休息天数可自定义；周期总长度等于当前上课周次数加连续休息天数。

## 主界面组件

- 插件注册“HL Education +”组件，可像 ClassIsland 自带“日期”组件一样添加到主界面。
- 正常上课显示当前周次和上课周期位置；休息日显示休息序号。
- 当天临时课表或休息日重复生效时，组件会显示对应的临时状态。

## 休息日处理

- 原定休息日启动时，如果当天没有有效临时课表且尚未处理，插件会询问“重复课表”或“忽略，保持休息”。
- 选择重复时可输入任意有效起始周，例如周十二。
- 重复从今天持续到当前连续休息段结束，依次使用周十二、周十三……；超过最后一周后回到周一。
- 当前休息段结束后，下一正常周期仍按原计划从周一开始，不会被临时重复改写。
- 选择忽略或直接关闭弹窗后，当天不再弹出；下一个原定休息日启动时会再次询问。
- 同样的操作入口会注入 ClassIsland 原“加载临时课表”抽屉，可随时改选重复开头或恢复休息。

## 课表优先级

插件完全接管时固定使用以下顺序：

1. ClassIsland 当天临时课表；
2. 当前休息段的临时重复；
3. 正常周期课表；
4. 原定休息并停用课表。

关闭插件或正常退出应用时，会恢复接管前的当天预定课表和课表总开关。插件不会删除原有非托管课表。

`ICycleCalendar.IsRestDay` 返回实际状态：当天临时课表或休息日重复生效时返回 `false`；`IsCycleActive` 表示课表完全接管是否正在运行，状态变化通过 `SettingsChanged` 通知集成的值日生服务。

## 班级批次时间表轮换

- 班级批次轮换会为同一套多周课程加载不同的时间表。
- 每个批次代表整个班级，不是学生个人；每个批次可绑定一份 ClassIsland 时间表。
- 可按日期号循环，也可校准今天使用哪个批次后按自然日连续循环。
- 插件会自动为旧配置补齐第一批和第二批时间表；两份时间表可独立修改时间线。

## Windows 定时关机

- 旧版独立定时关机模块已整合到 HL Education + 设置页，不再需要单独安装第二个插件。
- 新安装默认暂停，可添加任意数量的每日关机计划；每项可自定义名称、执行时间和启用状态。
- 所有关机计划共用 1–3600 秒倒计时；列表为空或所有计划暂停时不会自动关机。
- 倒计时覆盖全部屏幕，点击任意屏幕或按 `Esc` 可取消；安全预览不会执行关机命令。
- 错过计划时间后不会补触发，倒计时结束后执行 Windows 正常关机且不使用强制关闭参数。
- 若检测到独立插件的旧配置，会自动复制到 HL Education + 配置目录；迁移后应卸载独立插件，避免重复调度。

## 大周值日生

- “大周值日生”保留独立设置页，并继续提供“今日值日生”主界面组件、定时提醒、语音播报及 Excel/文本名单导入。
- 全新配置默认暂停；第一次成功点击“应用名单与人数”后会自动开启。后续手动暂停不会被名单修改自动重新开启。
- 值日生仅在实际的上课日推进；休息日不消耗名单，当天临时课表生效时会正常安排。
- 暂停 HL Education + 的课表完全接管后，值日生同步停止推进和提醒。恢复时从当天继续，不会补排暂停期间的日期。
- 首次运行 `2.0.0.0` 时，若 `cn.haigao.schedule104/duty-roster.settings.json` 不存在，会复制旧 `cn.haigao.duty-roster/settings.json`；目标配置优先，旧文件不会被覆盖或删除。
- 迁移会保留启用状态、提醒时间、语音设置、名单、轮次和当天安排。安装后应停用或移走旧独立值日生插件，避免重复轮换与提醒。

## 使用方法

1. 安装 `ClassIsland.HaiGao104.cipx` 并重启 ClassIsland。
2. 打开“应用设置”中的“HL Education +”。
3. 点击“打开原课表周期编辑器”，管理周次并填写各周课程。
4. 在同一编辑器顶部设置“今天是周几”，并在设置页填写连续休息天数。
5. 如需处理原定休息日，可使用启动弹窗或“加载临时课表”中的“HL Education + 休息日处理”。
6. 如需值日生轮换，打开独立的“大周值日生”设置页导入名单，并在主界面添加“今日值日生”组件。
