# HL Education + 定时关机

> 此功能已合并到“HL Education +”`1.9.0.0`，本项目仅保留共享源码与测试，不再单独生成或安装 `.cipx`。

ClassIsland 2.x 的 Windows 定时关机插件。插件每天在设置的时间显示全屏倒计时：

> 将在 xx 秒关机，点击屏幕任意处取消

## 安全设计

- 新安装后默认停用，默认时间为 22:00，默认倒计时为 60 秒。
- 倒计时可设置为 1–3600 秒。
- 鼠标或触摸点击任意屏幕、按下 `Esc`，都会取消本次关机。
- 错过计划时间后不会补触发，下一次执行时间自动顺延到次日。
- 使用 Windows 正常关机，不强制关闭可能包含未保存内容的程序。
- 设置页中的“安全预览倒计时”不会执行关机命令。

设置保存在 ClassIsland 的插件配置目录中，不会写入插件安装目录。
